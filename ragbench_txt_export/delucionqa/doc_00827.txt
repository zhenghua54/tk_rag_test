To Activate/Deactivate Push and release the Adaptive Cruise Control (ACC) on/off button.  The ACC menu in the instrument cluster display will read “ACC Ready.” To turn the system off, push and release the Adaptive Cruise Control (ACC) on/off button again.  At this time, the system will turn off and the instrument cluster display will read “Adaptive Cruise Control (ACC) Off.” WARNING: Leaving the Adaptive Cruise Control (ACC) system on when not in use is dangerous.  You could accidentally set the system or cause it to go faster than you want.  You could lose control and have a collision.  Always leave the system off when you are not using it.
