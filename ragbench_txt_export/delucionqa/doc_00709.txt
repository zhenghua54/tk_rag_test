Screen setup push and release the up or down arrow button until the screen setup menu icon/title is highlighted in the instrument cluster display.  Push and release the ok button to enter the submenus and follow the prompts on the screen as needed.  The Screen Setup feature allows you to change what information is displayed in the instrument cluster as well as the location in which that information is displayed.  Note: The Screen Setup feature can only be used while the vehicle is not in motion.  Note: Based upon equipment options and current vehicle status, some of the features may not be available.
