In Excel 2003, right-click the chart and choose Format Object. To assign the same size dimensions to all of the charts at the same time, do the following: 1  Select all of the charts by holding down the [Shift] key as you click each chart to create a multiple selection. To run the macro, return to Excel and select the sheet with the charts you want to resize. Click the Develop tab and then choose Macros in the Code group. Choose ResizeCharts() and click Run.
