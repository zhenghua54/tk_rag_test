Packages originally targeted for commercial versions of UNIX may attempt to install in the. Your general installation procedure will therefore be: 1  Read the README file and other applicable docs. 2  Run xmkmf-a, or the INSTALL or configure script. 3  Check the Makefile. 4  If necessary, run make clean, make Makefiles, make includes, and make depend. 5  Run make. 6  Check file permissions. 7  If necessary, run make install.
