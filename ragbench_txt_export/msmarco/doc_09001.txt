Data types. Think of a field's data type as a set of qualities that applies to all the values contained in the field and that determines what kind of data those values can be. For example, values that are stored in a Text field can contain only letters, numbers, and a limited set of punctuation characters. For example, a field whose data type is Text can store data that consists of either text or numerical characters, but a field whose data type is Number can store only numerical data. A field's data type determines many other important field qualities, such as: How you can use the field in expressions.
