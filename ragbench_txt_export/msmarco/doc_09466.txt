advertisement. Using Keyboard Shortcuts in MS Word You can avoid reaching for the mouse if you are familiar with keyboard shortcuts. If you forget one of these, use the mouse and go to the menu bar. In each pull down menu you will see keyboard commands given in the right side of the window. Commonly used keyboard combinations To use one of these combinations Hold the Ctrl or Alt key down and strike the letter key Ctrl+N Open a new word document quickly. Ctrl+X Cut-Removes the selection from the active document and places it on the clipboard. Ctrl+O Opens a previously saved document.
