A module for converting numbers or color arguments to RGB or RGBA. RGB and RGBA are sequences of, respectively, 3 or 4 floats in the range 0-1. This module includes functions and classes for color specification conversions, and for mapping numbers to colors in a 1-D array of colors called a colormap. colors. a list of matplotlib color specifications, or an equivalent Nx3 or Nx4 floating point array (N rgb or rgba values). name. a string to identify the colormap. N. the number of entries in the map. The default is None, in which case there is one colormap entry for each element in the list of colors.
