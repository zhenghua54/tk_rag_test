Dispatching Solutions. TTMS is a web based solution allowing users to work from anywhere with an internet connection. Some of the features offered are dispatching, invoicing, customer management, rates management, inventory management, payroll, sales reports as well as many other features. TTMS is built using the latest technology using the Micorsoft.NET framework for web site development and SQL Server for database management. TTMS offers integration to 3rd party eLog solutions, such as XRS, for drivers. TTMS provides the ability to see your drivers on a map without having to log into your eLog software and to view their available drive time and duty time hours.
