Click the Install button and Boot Camp Assistant will resize your Mac OS X partition and create the new partition for Windows. Your Mac will then restart and boot the Windows installer from the USB drive. Go through the installer and install Windows on your Mac like you’d install it on any other PC. However, if you already have a Boot Camp USB drive or you’ve already partitioned your Mac, you can uncheck these options to speed the process up.
