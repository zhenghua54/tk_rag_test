Make a linear segmented colormap with name from a sequence of colors which evenly transitions from colors[0] at val=0 to colors[-1] at val=1. N is the number of rgb quantization levels. Alternatively, a list of (value, color) tuples can be given to divide the range unevenly. colors. a list of matplotlib color specifications, or an equivalent Nx3 or Nx4 floating point array (N rgb or rgba values). name. a string to identify the colormap. N. the number of entries in the map. The default is None, in which case there is one colormap entry for each element in the list of colors.
