For a greater range of colors, you have two options. You can specify the color using an html hex string, as in: or you can pass an R, G, B tuple, where each of R, G, B are in the range [0,1]. colors. a list of matplotlib color specifications, or an equivalent Nx3 or Nx4 floating point array (N rgb or rgba values). name. a string to identify the colormap. N. the number of entries in the map. The default is None, in which case there is one colormap entry for each element in the list of colors.
