On the File menu, point to Export To, and then click PDF or XPS. In the File Name list, type or select a name for the form. In the Save as type list, click PDF. If you want to open the file immediately after saving it, select the Open file after publishing check box. Save as PDF for sharing or Web distribution. If you want to share the publication in a fixed format over the Web or a network, do the following: On the File menu, click Publish as PDF or XPS. In the File name list, type or select a name for the publication
